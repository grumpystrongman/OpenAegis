# OpenAegis Trademark Policy

OpenAegis code is open-source under AGPL, but the OpenAegis name, logo, and marks are brand assets owned by the project owner.

## Allowed

- truthful references such as "built on OpenAegis"
- internal use of the name/logo while evaluating or running unmodified OpenAegis

## Not Allowed Without Written Permission

- presenting a fork as the official OpenAegis project
- rebranding a derivative commercial service as "OpenAegis Cloud" or similar
- using the OpenAegis logo in ways that imply sponsorship or endorsement

## Purpose

This separates open code rights from brand protection, so the project stays freely usable while preventing brand capture.
